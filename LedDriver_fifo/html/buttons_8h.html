Built in and External Buttons initialization function with interrupts.

Initializes the board buttons by using a byte for these porpouse, button 1 match to byte=1, button 2 match to byte=2, ect

Parameters
[IN]struct device *gpio0 -> the pointer for gpio0 struct
[IN]char n_buttons -> byte corresponding to the buttons to activate
[IN]char c -> 'H' or 'h' to interrupt at rising edge -> 'L' or 'l' to interrupt at falling edge -> 'B' or 'b' to interrupt at both edges
